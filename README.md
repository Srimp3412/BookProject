ここからは制作途中の「Tundoku」というアプリの説明になります。
「Tundoku」というアプリは積読をする人向けに制作するアプリです。
具体的には積読している冊数や日数、読了した本の情報を友達に共有できるようにするという機能を搭載したいと考えています。
3月中には作り終えたいと考えています。
